## Принятые решения (что НЕ обсуждать)

Эти решения зафиксированы. Изменение требует нового ADR + согласования Архитектора.

1. **Залога нет.** Сервисный платёж невозвратный, при заезде, на расчётный счёт reHome.
   Договор найма и оферта требуют правок (блокер MVP).

2. **Финансовая модель коллаборантов утверждена** (ПЗ БЗ раздел 10.3):
   - A: payment_partner, kyc_provider, sms_voice, it_infrastructure, edo_provider, legal_consultant
   - B: cleaning, moving, key_delivery, repair_handyman (с эскроу и комиссией 10-25%)
   - C: insurance (referral)
   - D: management_company, emergency_service

3. **Уровень кабинета коллаборанта** выбирает сам коллаборант (по умолчанию LIGHT).
   УК/ТСЖ и аварийки автоматически в NONE.

4. **Двухконтурность фильтруется на уровне хранилища**, не приложения.

5. **scope** из JWT, не от клиента.

6. **REST + OpenAPI 3.1.** Источник истины — `01_postanovka/04_openapi.yaml`.

7. **53 эндпоинта в v1.0.** Реализация поэтапная (E1-E7), но контракт финальный.

8. **Технологический стек** утверждён: Django, FastAPI, Next.js, Postgres + pgvector,
   Qdrant, MinIO, Redis, Keycloak, Dramatiq.

## Глоссарий ключевых терминов

| Термин | Что значит |
|---|---|
| **kb-модуль** | Модуль базы знаний reHome — этот проект |
| **rehome.one** | Основная платформа, потребитель kb-API |
| **scope** | Вычисленная роль пользователя для текущего запроса |
| **access_level** | Уровень доступа ресурса: PUBLIC/LOGGED/AGENT/STAFF/LEGAL/HR_RESTRICTED |
| **Двухконтурность** | Разделение публичный/внутренний на уровне данных |
| **Сервисный платёж** | Невозвратный платёж нанимателя платформе при заезде. Замена залога |
| **Номинальный счёт** | Эскроу-счёт по ст. 860.1 ГК РФ для расчётов нанимателя-собственника |
| **Расчётный счёт** | Собственный счёт ООО «РЕХОМ» для дохода платформы |
| **Коллаборант (Collaborator)** | Внешний исполнитель: УК, клининг, страховая, банк-партнёр и т.д. |
| **PremisesCollaborator** | Связь объект ↔ коллаборант (роль, приоритет) |
| **ServiceOrder** | Заказ услуги у коллаборанта группы B с эскроу |
| **RAG** | Retrieval-Augmented Generation — паттерн AI-чата |
| **LLMProvider** | Абстракция над AI-провайдерами для переключения без рефакторинга |
| **Eval-стенд** | Сервис прогона тестовых запросов для выбора LLM |
| **m2m** | Machine-to-machine аутентификация (Bearer JWT через Client Credentials) |
| **Phase 0** | Инвентаризационная фаза проекта (state-of-code.md) |
| **ADR** | Architecture Decision Record — фиксация архитектурного решения |
| **DoD** | Definition of Done — критерии готовности задачи |
| **Однофазный коммит** | Упрощённый merge (≤3 файла, ≤100 строк, не критические пути) |
| **Двухфазный коммит** | Стандартный merge с раундами правок |
| **ФЗ-152** | Федеральный закон о персональных данных |
| **КЭП** | Квалифицированная электронная подпись (63-ФЗ) |
| **КЭДО** | Кадровый ЭДО (Контур.Диадок, HRlink) |
| **Идемпотентность** | Повторный запрос даёт тот же результат (Idempotency-Key) |
